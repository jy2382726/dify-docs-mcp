## 2. 三层架构说明

### Layer 1：文档搜索（免费 — Mintlify MCP）

| 项目 | 内容 |
|------|------|
| 来源 | Mintlify 为 docs.dify.ai 自动生成的原生 MCP Server |
| 端点 | `https://dify-6c0370d8.main-kill-isr.mintlify.me/mcp` |
| 发现方式 | `https://docs.dify.ai/.well-known/mcp.json` |
| 认证 | 无（公开文档） |
| 传输 | Streamable HTTP |
| 速率限制 | 5,000 请求/小时/用户，1,000/小时/站点 |

**提供的工具：**

| 工具 | 功能 | 参数 |
|------|------|------|
| `search_dify_docs` | 语义搜索全部文档 | `query`（必填）、`version`、`language` |
| `query_docs_filesystem_dify_docs` | 虚拟文件系统直接读取页面 | `command`：支持 `rg`/`cat`/`tree`/`head`/`grep` 等 shell 命令 |

**覆盖的文档内容：**

| 分类 | 页面数 | 路径 |
|------|--------|------|
| 节点文档 | 21 个 | `/en/use-dify/nodes/*.mdx`（llm、code、if-else、iteration、loop 等） |
| 构建指南 | 11 个 | `/en/use-dify/build/*.mdx`（workflow-chatflow、agent、mcp 等） |
| 入门教程 | 3 个 | `/en/use-dify/getting-started/*.mdx` |
| 工作流教程 | 6+ 个 | `/en/use-dify/tutorials/workflow-101/*.mdx` |
| 调试指南 | 4 个 | `/en/use-dify/debug/*.mdx` |
| 发布指南 | 5 个 | `/en/use-dify/publish/*.mdx` |
| API 文档 | 20+ 个 | `/api-reference/*.mdx` |
| 插件开发 | 20+ 个 | `/en/develop-plugin/*.mdx` |
| OpenAPI | 有 | `/openapi/` |

**已验证的文档质量：**
- LLM 节点：覆盖模型选择、Prompt 配置、Context RAG、结构化输出、Memory、Vision、Jinja2、错误处理
- Code 节点：覆盖 Python/JS、输入输出、沙箱安全、依赖支持、输出限制
- Variable Assigner：覆盖 5 种数据类型操作、Smart Memory 模式
- Workflow vs Chatflow：覆盖区别、节点系统、触发方式
- 变量引用：文档中使用 `{{variable_name}}` 格式（Prompt 模板层）

## 4. 已验证结论汇总

### 4.1 技术事实（源码验证 ✅）

| 结论 | 验证方式 | 置信度 |
|------|---------|--------|
| 核心节点迁移至 graphon 包（`langgenius/graphon`，PyPI: `graphon==0.4.0`） | node_factory.py 注册机制 + graphon 仓库结构 | 高 |
| 变量引用格式为 `{{#node_id.variable#}}`（含 `#`） | 前端 VAR_REGEX 正则 + 后端变量选择器 | 高 |
| 变量前缀：sys / env / conversation / rag | `variable_prefixes.py` 源码 | 高 |
| 系统变量 16 个（query, files, user_id 等） | `system_variables.py` 源码 | 高 |
| Dify 文档由 Mintlify 托管 | `.well-known/mcp.json` 返回 Mintlify 端点 | 高 |
| graphon 包 0.4.0，"still evolving" | PyPI 页面描述 | 高 |

### 4.2 产品事实（对抗评估 + 实测）

| 结论 | 来源 | 置信度 |
|------|------|--------|
| 200 页文档不值得建 RAG 系统 | 红队质疑 1.2，未被有效反驳 | 高 |
| Mintlify MCP 免费可用且功能强大 | 实测验证（2 个工具、21 个节点页面） | 高 |
| yzmw123 Skill 覆盖 80%+ 需求 | 源码分析（25+ Schema、校验脚本、4 模板） | 高 |
| 官方 dify-docs-mcp-server 5 个月归档 | GitHub 记录 | 高 |
| B→A 渐进路径风险最低 | 集成评估师评分 8.0/10 | 中高 |

### 4.3 待验证项

| 项目 | 优先级 | 验证方式 |
|------|--------|---------|
| Mintlify MCP 的变量引用覆盖度（DSL 层 `{{#...#}}` 格式） | 中 | 实测：用 `rg "{{#" /en/` 搜索 |
| graphon 0.4.0 → 0.5.0 的 breaking change 风险 | 中 | 关注 graphon GitHub releases |
| Skill 在实际使用中的覆盖率 | 中 | 团队试用 2-4 周后收集反馈 |
| Dify 文档覆盖的最新版本号 | 低 | 用 `search_dify_docs(query="release changelog")` 测试 |
